# B360-Aorus-Gaming3
### Features
- OpenCore & Clover
- Full-featured: Sleep/Wake, Audio, USB 3.0/2.0, Webcam (External), ...;
- Compatible with all macOS version (From Mojave to latest version)

Specs
- CPU: Intel 9400F
- Main : B360 Aorus Gaming3
- Vga: Rx 560 4GB
- Ram 16GB DDR4-2666
- SSD NVMe Sam Sung SM961 256GB
- Wifi: BCM943602CS (native)
